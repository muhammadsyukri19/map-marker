## ✨ Fitur yang Dikembangkan
Aplikasi ini memetakan **Masjid Raya Baiturrahman, Banda Aceh** sebagai titik pusat referensi, serta dilengkapi berbagai fitur tambahan:
1. **Modern UI & Full-Screen Map:** Tampilan peta penuh menutupi layar dengan antarmuka bergaya *Glassmorphism* dan ikon profesional (Font Awesome).
2. **Pencarian Lokasi Cerdas:** Pengguna dapat mengetikkan nama suatu tempat dan peta akan otomatis terbang ke lokasi tersebut (terintegrasi API Nominatim OpenStreetMap).
3. **Deteksi Lokasi Pengguna (Geolocation):** Menampilkan posisi koordinat asli pengguna (GPS) saat ini beserta tingkat akurasinya.
4. **Modal Info Kelengkapan Kelompok:** Menampilkan pop-up interaktif berisi data mahasiswa pembuat project.

## 🚀 Cara Menjalankan
1. Clone / unduh repository ini ke komputer Anda.
2. Buka folder `kelompok-9`.
3. Buka (double-click) file `index.html` menggunakan browser standar (Chrome, Edge, Firefox, dll), atau gunakan ekstensi **Live Server** pada VS Code. Tidak memerlukan konfigurasi server tambahan.
